# HOME

A web development project with Sass compilation and live server capabilities.

## Installation

Install all dependencies:

```bash
npm install
```

## Available Commands

### Development Server
Start a live development server:
```bash
npm run serve
```

### Sass Compilation
Compile and watch Sass files (automatically recompiles on changes):
```bash
npm run compile:sass
```

## Project Structure

- `indec.html` - Main HTML file
- `scss/` - Sass source files
- `css/` - Compiled CSS files
- `images/` - Image assets

## Development Workflow

1. Install dependencies: `npm install`
2. Start Sass compilation in one terminal: `npm run compile:sass`
3. Start live server in another terminal: `npm run serve`
